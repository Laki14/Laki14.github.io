Introduction

CSS Grid Layout is the most powerful layout system available in CSS. It is a 2-dimensional system, meaning it can handle both columns and rows.

Unlike Flexbox, which is largely 1-dimensional, Grid excels at aligning content in two dimensions simultaneously.

display: grid;
Grid Container

To get started you have to define a container element as a grid with the display property.

Common properties for containers include:

  • grid-template-columns
  • grid-template-rows
  • grid-gap
  • justify-items
  • align-items
.container { display: grid; }
Grid Tracks

Tracks are the space between any two lines on the grid. They are essentially the columns and rows of your layout.

You define tracks using the grid-template-columns and grid-template-rows properties.

Once you define your tracks, every child element can be placed with precision by referencing the numbered lines around those tracks.

grid-template-columns: 200px 1fr;
Grid Lines

The dividing lines that make up the structure of the grid are called Grid Lines.

They can be vertical (column lines) or horizontal (row lines). You can refer to them by number to place items.

Named lines can also make the layout easier to read when you are building larger page systems with repeated placements.

grid-column-start: 1;
The Fr Unit

The new fr unit represents a fraction of the free space in the grid container.

This allows you to set the size of tracks in a way that is flexible and responsive without using percentages.

  • Use fr units to divide remaining space proportionally.
  • Combine fr with fixed sizes when one column must stay stable.
  • Pair fr units with minmax() for stronger responsive layouts.
grid-template-columns: 1fr 1fr 1fr;
