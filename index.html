HTML vs CSS

HTML is simple — it has about 100 elements, and you can do most things with only half of them.

CSS is much more complex — it controls colors, fonts, layouts, and almost everything visual.

Browsers have default styles: black text, white/grey background, Times New Roman font.

2. Design Basics (CRAP)

  • C: Contrast – make text and background easily readable.
  • R: Repetition – repeat design elements for consistency.
  • A: Alignment – line up items neatly.
  • P: Proximity – group related items together.

Start simple with CSS. Changing color, background, and font can make a huge difference.

3. Choosing Colors

  • Use coolors.co to pick colors that go well together.
  • Save the URL — it contains your color choices for later reference.
  • Make sure foreground and background colors have good contrast.
  • Consider using different colors for headers, footers, and main content.

4. Choosing Fonts

  • Small text → sans-serif (clean, readable).
  • Large text → can use serif or stylized fonts.
  • Pick at least two fonts:
    • One for normal page text
    • One for headings (like h1)
  • Use Google Fonts for variety and include them properly in your site.

5. Margins & Page Layout

  • Adjust margins and max-width to control where content sits.
  • Example: max-width: 900px; margin: 0 auto;
  • Keeps content centered on the screen without centering paragraphs or bullets.
  • Test on different screen sizes to make sure it looks good.
